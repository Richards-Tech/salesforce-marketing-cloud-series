## ⚡ Triggered Sends vs User-Initiated Sends in Salesforce Marketing Cloud

When working with real-time and batch communication in SFMC,  
understanding these two concepts is critical for architecture & interview clarity.  

---

### 🔹 1. Triggered Sends (Real-Time Communication)

👉 **Definition:**  
Emails sent automatically when a specific event occurs.  

💡 **Simple Understanding:**  
Triggered Send = “System reacts instantly to an action”  

📌 **Examples:**  
- Welcome Email after signup  
- OTP / Password Reset  
- Order Confirmation  

🔧 **How it Works:**  
- Uses Triggered Send Definition  
- API / Event-based execution  
- No manual scheduling required  

🧠 **Key Features:**  
✔ Real-time delivery  
✔ Highly personalized  
✔ Used in transactional journeys  

---

### 🔹 2. User-Initiated Sends (Batch Communication)

👉 **Definition:**  
Emails sent manually or scheduled by a marketer to a group of users.  

💡 **Simple Understanding:**  
User-Initiated = “Marketer decides when to send”  

📌 **Examples:**  
- Promotional Campaigns  
- Newsletters  
- Festival Offers  

🔧 **How it Works:**  
- Created in Email Studio  
- Sent via Send Flow / Automation / Journey  

🧠 **Key Features:**  
✔ Scheduled or manual  
✔ Sent to large audience (DE / List)  
✔ Used for marketing campaigns  

---

### 🔥 Key Differences (Interview Ready)

| Feature        | Triggered Send        | User-Initiated Send   |
|----------------|-----------------------|-----------------------|
| Trigger        | Event-based           | Manual / Scheduled    |
| Speed          | Real-time             | Batch                 |
| Use Case       | Transactional         | Promotional           |
| Personalization| High                  | Moderate              |
| Setup          | API / Triggered Definition | UI / Automation |

---

### 🎯 When to Use What?

👉 **Use Triggered Sends when:**  
- Immediate response is required  
- Event-based communication  
- Critical user actions  

👉 **Use User-Initiated Sends when:**  
- Campaign planning is needed  
- Bulk audience targeting  
- Scheduled marketing activities  

---

### 💡 Key Takeaway
Triggered Sends = Real-time, transactional communication  
User-Initiated Sends = Batch, promotional communication  

👉 Knowing the difference ensures correct architecture, better campaign design, and strong interview answers.
